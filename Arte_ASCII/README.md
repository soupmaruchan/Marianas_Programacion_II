## ꒰ᐢ. ̫ .ᐢ꒱ 
Menú con diversas funciones que generan arte ASCII con el que 
el usuario podrá interactuar y a su vez visualizar tu arte
realizado dentro de la galería. Algunos pictogramas contarán 
con simulación de movimiento y una historia que integre las 
funcionalidades del arte ASCII en Python.

## ꒰ᐢ. ̫ .ᐢ꒱ Conceptos utilizados
★ Variables y strings - La modificación de los mismos permite una
visualización óptima del arte ASCII a través del formateo.
★ Loops - La anidación de los mismos permite generar bucles para 
determinar cuándo dejar de reproducir cierta funcionalidad.
★ Range - Señalización de parámetros.
★ Colores ANSII - Agrega colores a la interfaz.

## ꒰ᐢ. ̫ .ᐢ꒱ ¿Cómo funciona?
La terminal abre un menú con los siguientes apartados:
 ∙ ~εïз~ ∙ 1. Patrones geométricos
 ∙ ~εïз~ ∙ 2. Banner
 ∙ ~εïз~ ∙ 3. Marcos
 ∙ ~εïз~ ∙ 4. Animaciones
 ∙ ~εïз~ ∙ 5. Tabla de multiplicar
 ∙ ~εïз~ ∙ 6. Tal vez pueda renacer en algo hermoso
 ∙ ~εïз~ ∙ 7. Ver galería
 ∙ ~εïз~ ∙ 8. Salir

 Al ingresar un número tendrás acceso a las funciones del cada
 respectivo apartado, todo lo generado por el usuario se almacena 
 para su posterior consulta.

✿ Patrones geométricos
El usuario podrá seleccionar las dimensiones de diferentes figuras
como triángulo, cuadrado y pirámide

✿ Banner
Al ingresar un texto este se podrá almacenar en un recuadro decorado.

✿ Marcos
Se podrá decorar el texto ingresado por el usuario con marcos florales,
de estrellas y de corazones.

✿ Animaciones
Se podrá seleccionar entre una animación de una barra de progreso y 
texto móvil en la terminal.

✿ Tabla de multiplicar
Generador de las tablas de multiplicar de forma estética.

✿ Tal vez pueda renacer en algo hermoso
Historia animada sobre una cucaracha que quisiera ser una mariposa
para que todos la traten bien.

✿ Ver galería
Se almacena todo el arte ASCII generado por el usuario.

✿ Salir
Se rompe el bucle y se cierra el programa.
